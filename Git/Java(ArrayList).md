# ArrayListとは
「可変長配列」を使用するための仕組み。   
可変長配列とは、文字通り長さ（要素数）を変更できる配列のこと

噛み砕くと・・・   
ArrayListはデータを入れるためのリストで、そのリストの中にデータを入れたり、取り出したり、削除したりすることができる。
そして、ArrayListは柔軟で、必要に応じてデータを変更できるのが特徴。
