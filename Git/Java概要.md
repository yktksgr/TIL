# クラス名の命名指針
Javaには クラス という概念があり、Java ファイル名(Sample.java)の「.java」を省いたものがクラス名と。クラス名には命名指針がある。   
Javaエンジニアは下記の命名指針に従ってファイルを作成する。

クラス名は大文字からはじまり各単語を大文字で連続するキャメルケース（UpperCamelCase）で命名する   
クラス名は大抵名詞もしくは名詞句ですが、場合によっては形容詞、形容詞句になる
